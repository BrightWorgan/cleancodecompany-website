Frequently Asked Questions

Here are some of our most asked questions
  • Do you manage domain hosting?

    Yes, we offer space on a shared server to reduce cost or a dedicated instance if you're anticipating high traffic.

  • What do I need to do to get a website off the ground?

    Websites are highly individual, what we need depends from client to client and we'll discuss exactlly what we need when we first get in contact. However, we usually need the following:

    • A clear goal for the website
    • Some links to websites that you like (to give us an idea of the theme you're aiming for)
    • Ideally a logo so we can always keep that in mind while working on your project

  • Can I see other websites you have previously done?

    We are always delighted to showcase our previous work when possible. Here are three examples of recent projects we've undertaken:

    • The Eating House
    • West Wickow Pool
    • Raheny Carpets

  • How long does a project take?

    There are several key steps that we take to produce a piece of work, be it a simple one-page website or a complex mobile application. It all starts with an intial client meeting, where meet the client and discuss options. After this, begins the design phase, where we create a deisgn plan for the project. Once we have a design, a mock up or first draft is created and we meet with the client once more to discuss the first draft and to get the all important client feedback. After any needed changes are implemented we launch the project. It is also important to note that if the project is on the more complex side, like a mobile application, we most likey will have multble design phases and draft meetings before the full launch as there is more items to create.

  • What technologies do you use?

    We use a wide array of tools and languages. Which one we pick depends on the project at hand, if you already have a codebase we're happy to work with nearly any language or framework.

    • HTML/CSS
    • NodeJS
    • ReactJS
    • React Native
    • Swift
    • Kotlin
    • Wordpress
